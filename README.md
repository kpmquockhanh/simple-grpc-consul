# simple-microservices

This is a simple/example for grpc call with consul as a service discovery

Using `protogen` to build proto

Run sample api with `hello-api`

Run sample dms/grcp-server with `sample-dms`

```
| - core 
| - api
| - services
| - proto
| - tools
```
